## 1. Why this project exists

This project asks what becomes operationally meaningful on the ecology side
once Graph Reflexive Coherence can carry bounded relations such as transport,
history, trace, susceptibility, restoration, and later eligibility.

It does not merely rename graph primitives with ecological words:

```text
graph project
  asks what the substrate can carry

agentic ecology project
  asks what new participant-relative, compositional, and ecological
  distinctions can be made once those substrate relations exist
```

The working spiral is bidirectional. RCAE first consumes suitable native LGRC
surfaces. When an ecology question requires a missing distinction, RCAE may
construct it explicitly, preserve its producer and naturalization debt, and
return a precise demand to the graph project. A later native implementation
never silently upgrades an earlier constructed result.

## 4. What the completed lanes show

### L01 — differential possibility

P2-I1 shows that attributable shared-medium history can interact with local
susceptibility so that matched participants receive different later formation
possibilities. Its novelty relative to N30 is selectivity:

```text
N30:   history changes later eligibility
L01:   the same history changes eligibility differently for different
       local differentiations
```

The claim ceiling is a bounded niche-conditioning demand pattern, not
autonomous niche formation.

### L02 — common-carrier composition

P2-I2 shows that several physical contributions can constitute one non-private
carrier whose current state, active history, or combined state/history governs
a later native response. The retained modes are not ranked.

Appendix A separately shows fixture-level complementarity among three physical
operations. Appendix B shows that apparently similar G/E/P behavior in active
history is instead a three-admission terminal-P temporal gate. Output language
alone therefore does not identify carrier mechanism.

The claim ceiling remains scaffold-dependent and operational. It is not a
general shared-pool ecology, cooperation, metabolism, or organization result.
