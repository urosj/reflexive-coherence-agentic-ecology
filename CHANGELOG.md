## [0.1] - 2026-06-18

### Added

- Added public-facing README, contribution guide, code of conduct, citation
  metadata, release notes, and publication checklist.
- Added `.gitignore` rules for caches, virtual environments, generated output,
  tool state, and secret-bearing local config.
- Added a repository-level claim boundary for the current conceptual/paper-only
  status.
- Added `From State to Becoming`, a transition essay for readers moving from
  state-driven control toward Reflexive Coherence becoming.
- Added `RC Agentic Ecology`, a specification draft for translating
  state-rule agents into RC agentic ecologies, with RC-Ant Colony as the first
  detailed worked example.
- Added `The Shared Medium`, a transition essay from direct message passing
  toward shared-medium participation, co-response, trace, resonance, and
  field-mediated coordination.
- Added `Shared-Medium Coordination`, a conceptual engineering specification
  for translating communication, signaling, stigmergy, and multi-agent
  relation into shared medium, perturbation, trace, pressure, susceptibility,
  co-response, resonance, and medium debt.
- Added README orientation sections for core translation, intended readers,
  practical reading modes, and maturity levels.
- Added GPL-2.0-only repository license text and CC BY-SA 4.0 paper headers.

### Changed

- Expanded citation, contribution, release-note, and publication-checklist
  metadata to cover shared-medium coordination, message scaffolding, and medium
  debt.
