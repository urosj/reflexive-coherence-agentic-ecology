# AE01 Selected Outputs

This directory is reserved for compact, reviewed, retained AE01 evidence after
execution. P1-I5 produces no selected evidential output here; its calibration
and developmental-interpretation fixtures validate infrastructure only.

Scratch runs, registered-probe transients, receipts not selected for evidence,
validation summaries, resolved-policy views, and duplicate reconstructions
belong under ignored top-level `outputs/`.

An output enters this directory only when its manifest is fully resolved, its
reconstruction passes, its evidence tier is `retained_evidence`, and its
retention decision is explicit. A retained lane result must include its frozen
metric sheet and candidate-blind calibration, per-seed margins, terminal
classification, linked developmental interpretation, and applicable runtime
receipt. Large local-only artifacts remain represented by verified
reconstructable manifest entries.

## P2-I3 B-R I05 calibration

`p2-i3/i05/` retains the consumed one-shot claim, successful final receipt,
five-case matched-null panel, metric calibration, frozen metric sheet, and
independent result validation. All ten `m_trace`/`m_export` null margins are
exactly zero, so the shared `delta` is the preregistered numeric floor
`1e-12`. The result is candidate-blind numeric-resolution evidence only and
is owner-accepted. `P2-I3-CAL-GATE` is passed and opens I06 exact registration
only.

The permanent claim prohibits rerun. Use the read-only reconstruction command
in `reports/P2-I3-I05-BR-calibration-result.md` with absent reconstruction
destinations.
