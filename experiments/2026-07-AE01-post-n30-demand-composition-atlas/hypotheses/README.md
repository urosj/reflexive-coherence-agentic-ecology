# AE01 Hypotheses

This directory owns preregistered AE01 lane and cross-lane hypotheses. Each
hypothesis must be frozen before its lane executes and must declare positive,
negative, blocked, and incomplete outcomes together with finite stopping
conditions.

The planned scope covers the seven initial lanes, recurring cross-lane
requirements, and explicit N31+ non-selection when no candidate passes the
ranking gate. Hypotheses must preserve the distinction between N29/N30 evidence,
ecology extrapolation, and constructed RCAE mechanisms.

No hypothesis is frozen and no result is recorded here yet.
