## 1. Purpose

This document transfers one precise substrate question from Agentic Ecology
back to the graph/LGRC project and records exactly how P2-I3 must resume after
that question is investigated.

The handoff exists because P2-I3 reached a genuine graph/ecology spiral point:

```text
N30 established a bounded trace-mediated shared-medium relation
-> P2-I3 asked for a non-static trail-field dynamic
-> Q-005 found that "decay" has no single admitted LGRC meaning
-> forcing a local RCAE choice would make ecology define substrate semantics
-> N31 should discriminate and, if warranted, naturalize the missing primitive
-> P2-I3 may then consume the bounded N31 result
```

This is not a failure of P2-I3. It is the intended downward part of the
cross-project spiral: ecology demand exposes a missing graph distinction, and
the graph project tests it without borrowing ecology demand as evidence.

## 4. Graph-side question selected by the demand pass

### 4.1 Recommended N31 identity

```text
N31 — LGRC9V3 Decay Semantics and Native Primitive
```

The existing N30+ candidate menu contains a slow-memory conductance-trace
direction. The later cross-project addendum deliberately leaves N31 unfixed
until Agentic Ecology supplies a demand. This handoff selects the broader
decay-semantic question rather than assuming conductance is already the
answer.

The graph project may refine the display name while preserving the returned
identity and scope.

### 4.2 Core question

```text
What LGRC9V3 state or relation can autonomously weaken under system-internal
time while preserving coherence, causal locality, restoration identity, and
composability; and which realization, if any, warrants admission as a native
decay primitive?
```

### 4.3 What “best” means

The best primitive is not the implementation with the largest numeric effect
or the easiest path to a positive result. N31 should assess:

- compatibility with RC theory and declared LGRC semantics;
- exact coherence conservation or an explicit non-coherence invariant;
- system-internal rather than wall-clock time;
- local causal action rather than global ranking;
- minimal hidden continuation state;
- complete snapshot/load/reset/branch identity;
- independently intervenable state and lifecycle;
- topology dependence and transfer scope;
- implementation simplicity without semantic relabeling;
- producer-to-native naturalization clarity;
- usefulness across later graph experiments and ecology composition; and
- honest claim ceiling when no universal primitive exists.

N31 may validly conclude that LGRC needs more than one primitive.

## 5. Candidate meanings N31 must preserve

### 5.1 Candidate A — release-efficacy attenuation

```text
carrier coherence remains
-> a registered internal age/phase changes future packet formation
-> newly created packets carry less coherence
```

Mandatory boundary:

- packet amount may be selected at creation;
- an in-flight packet may not attenuate;
- source debit and target credit use the same created amount;
- unreleased coherence remains at the source; and
- any age/phase state is explicit continuation state.

N31 should determine whether this is a true field-decay primitive or only a
response/expression policy. A valid result may reject it as primitive decay
while retaining it as a useful response-control pattern.

### 5.2 Candidate B — coherence-conserving source leakage

```text
source internal coherence drives bounded emission
-> native packet departure debits the source
-> in-flight state carries the exact amount
-> a declared destination receives the exact amount
-> local source field magnitude decreases
```

Mandatory boundary:

- local loss has an explicit destination;
- total node-plus-in-flight coherence remains invariant;
- emission uses internal LGRC time and source state;
- the receiver cannot be omitted from topology or continuation identity;
- lifecycle packets remain distinct from scientific probe packets; and
- producer scheduling cannot precompute the desired trajectory.

N31 should determine whether an endogenous emission/leakage operation is the
minimal native decay primitive, a special transfer policy, or insufficiently
general.

### 5.3 Candidate C — route-susceptibility relaxation

```text
attributable activity updates a route-local relational/coupling state
-> that state affects later local transport
-> without reinforcement it relaxes toward baseline
```

Mandatory boundary:

- current theory supports instantaneous state/flux-derived geometry but does
  not already define slow susceptibility;
- N08 records native route-conductance memory as missing;
- N22 supplies producer-mediated conductance precedent, not native authority;
- susceptibility is not coherence mass and needs its own bounds/invariants;
- its update, cost, relaxation, and coupling law must be explicit; and
- it may not be an opaque score read by a global router.

N31 should determine whether this relational state can be theory-authorized
and naturalized, remains only a constructed mechanism, or must be rejected.

### 5.4 Candidate-specific topology

N31 must not force the candidates into one topology merely to compare them.

```text
release attenuation may require a local expression/receiver relation
leakage requires a closed destination/reservoir relation
susceptibility may require edge/corridor coupling and local conductance
```

Each candidate receives the smallest topology that faithfully expresses its
carrier, invariant, local action, and restoration state. Cross-candidate
comparison should use semantic criteria and normalized evidence roles, not
raw effect magnitudes from nonequivalent fixtures.

## 6. N31 minimum experimental boundary

N31 should remain a small graph-primitive experiment. It should not reproduce
P2-I3's complete two-route ecological program.

### 6.1 Required common relation

Every candidate claimed as decay must demonstrate:

```text
declared state or relation formed
-> forming activity stops
-> system-internal progression occurs
-> declared state or relation changes autonomously
-> a local readout changes because of that state
-> intervention on the mediator changes/removes the readout effect
-> complete state restores and continues identically
```

### 6.2 Required distinctions

N31 must separate:

- persistence from decay;
- field-state decay from expression attenuation;
- redistribution from destruction;
- instantaneous geometry from slow relational aftereffect;
- internal time from wall time;
- native transition from producer policy;
- current-state sufficiency from hidden active history;
- local encounter from route ranking;
- implementation conformance from scientific evidence; and
- candidate usefulness from native admission.

### 6.3 Minimum controls

Control mechanics may differ by candidate, but N31 must resolve these intents:

- no forming activity;
- formation present but lifecycle disabled;
- lifecycle active with mediator clamped;
- current state installed without formation history;
- complete-state-matched history contrast where causally meaningful;
- time/phase intervention;
- destination/accounting audit for any moved coherence;
- local-readout versus global-selector exclusion;
- snapshot/load/branch equal continuation;
- producer absent or replaced by native operation; and
- candidate-specific topology symmetry or bounded asymmetry.

### 6.4 Valid outcomes

N31 must allow:

```text
one native decay primitive selected
multiple distinct native primitives selected
producer-mediated candidate only
expression attenuation classified below decay
instantaneous geometry classified below durable decay
no candidate admitted
new missing distinction exposed
```

Selection is not mandatory.
