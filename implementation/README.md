# Implementation Directives

This directory contains cross-cutting project architecture, implementation
plans, decision records, checklists, and handoffs. It does not contain
experiment evidence or canonical behavioral specifications.

## Active Post-N30 directives

- [Post-N30 project architecture, atlas, specification, and implementation plan](PostN30-plan.md)
- [Post-N30 master program checklist](PostN30-checklist.md)

The plan controls rationale, boundaries, decisions, and sequencing. The
checklist is the authoritative progress view. Experiment-local implementation
records belong under their owning experiment and must cite these directives.

Accepted decisions change only through recorded revision. A document in this
directory cannot raise an experiment claim, admit a catalog item, or authorize
modification of the graph/PyGRC repository.
